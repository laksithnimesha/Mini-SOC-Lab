\# Project Overview



The Mini SOC Lab is a practical security monitoring environment built to simulate, detect, and investigate network attacks in a controlled lab.



This project was designed to demonstrate core SOC concepts including:



\- attack simulation

\- endpoint monitoring

\- network intrusion detection

\- SIEM alerting

\- packet-level investigation



The lab uses Kali Linux as the attacker, Ubuntu as the monitored target, and Wazuh as the central SIEM platform.



Suricata provides network intrusion detection, while tcpdump and Wireshark provide packet-level evidence for investigation.



The project simulates common attack scenarios such as:



\- reconnaissance (Nmap)

\- SSH brute-force (Hydra)



These attacks are detected through:

\- endpoint logs

\- IDS alerts

\- SIEM correlation

\- packet capture validation



This project demonstrates practical blue-team workflow and analyst-style investigation in a self-hosted lab.
